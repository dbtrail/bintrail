# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in bintrail, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email: **daniel@dbtrail.com**

Include:
- A description of the vulnerability
- Steps to reproduce
- Affected versions (if known)
- Any potential impact

You should receive a response within 72 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Supported Versions

Security fixes are applied to the latest release only.

| Version | Supported |
|---|---|
| Latest release | Yes |
| Older releases | No |
